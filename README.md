# Round Robin CPU Scheduling Simulator

This project simulates the Round Robin CPU Scheduling Algorithm.

## Features
- Calculates Completion Time (CT)
- Turnaround Time (TAT)
- Waiting Time (WT)
- Generates Gantt Chart
- Dynamic arrival and burst times

## How to Run
g++ round_robin.cpp -o rr
./rr

## Example Input
Enter number of processes: 3
Enter time quantum: 2
P1 → AT=0 BT=5
P2 → AT=1 BT=3
P3 → AT=2 BT=4

C++
